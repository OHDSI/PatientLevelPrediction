# join two lists

join two lists

## Usage

``` r
listAppend(a, b)
```

## Arguments

- a:

  A list

- b:

  Another list

## Value

the joined list

## Details

This function joins two lists

## Examples

``` r
a <- list(a = 1, b = 2)
b <- list(c = 3, d = 4)
listAppend(a, b)
#> $a
#> [1] 1
#> 
#> $b
#> [1] 2
#> 
#> $c
#> [1] 3
#> 
#> $d
#> [1] 4
#> 
```
