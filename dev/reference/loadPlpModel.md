# loads the plp model

loads the plp model

## Usage

``` r
loadPlpModel(dirPath)
```

## Arguments

- dirPath:

  The location of the model

## Value

                        The plpModel object

## Details

Loads a plp model that was saved using
[`savePlpModel()`](https://ohdsi.github.io/PatientLevelPrediction/dev/reference/savePlpModel.md)
